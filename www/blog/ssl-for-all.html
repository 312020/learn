    SSL now available to all

    As the web moves towards SSL everywhere, JS Bin has released this feature from Pro only accounts and made it available to everyone as of today.

    This means that you can swap http for https and JS Bin will load up. Remember there are considerations with SSL.

    If you're stickler for security, registered users can opt-in to default to SSL via the account preferences.

    You do need a pro account to embed bins over SSL, and this will remain a pro feature.

    What this does mean is that I've started to lay the foundations to add offline support via Service Worker, so stay tuned!
